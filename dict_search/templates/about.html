About LiODin

Linked Open Dictionary Navigator is a web application that allows users to load and browse OntoLex-lemon dictionaries modelled in the Linked Data paradigm.


We support several dictionaries right now, but more importantly, users can specify their own endpoint and use it alongside all the dictionaries we included.


This tool is the first universal user-friendly interface for the vast ocean of both retrodigitised converted dictionaries and RDF-first dictionaries of different types.
